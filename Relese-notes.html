Tuesday Dec 6, 2022
Build number: 30.0
ENHANCEMENT
Fiscal Printer display [Italy]
Fiscal Printer display now shows the real time order value for Italy.
ENHANCEMENT
Additional check on generation of Invoices [Italy]
An Additional check is implemented on the view confirmation page to disallow user from generating an invoice and Tax Free without printing the receipt.
ENHANCEMENT
Fixed POS - Automatic scan on entering the cart
On the Fixed POS, when opening a new cart, the user can start scanning without the additional step of first clicking on the scan field.
ENHANCEMENT
Order Lines
The Order Lines page has two new changes:
  • Date Headers are now added
  • Quantity is moved to the right under the price
FEATURE
Print Queue [Italy]
Print Queue:
  • The print receipt button on the view confirmation is now disabled for Italy.
  • A new button is added in the burger menu, where the user can go to check all the receipts in errors and then decide if they need to print the receipt or just enter the fiscal number for that order.
FEATURE
New Store Close
  • Italy Stores:
    • The user can now reopen the till and close the till any number of times. (Multiple Till Closure).
    • The user will not be able to close the till without printing all the receipts that are in error.
    • A new store totals page is implemented in the store close that gives better visibility into the payment totals for the day.
  • Non - Italy Stores:
    • A new store totals page is implemented in the store close that gives better visibility into the payment totals for the day
FEATURE
Order Timeline
A new feature is added onto the contextual menu in the order profile where the user can check the history of changing customers and sales associate post sale. This Feature can be accessed by only the Managers.
FEATURE
Foreign Currency
A new payment option called Foreign Currency is added in the payments page, where a customer can pay with foreign currencies that are configured for that store
BUG FIX
Scanning speed issue in the cart on the Fixed POS
Scanning items in the cart is now optimized to be much faster on the Fixed POS
Tuesday Oct 25, 2022
Build number: 29.0
ENHANCEMENT
Latest iOS version 16 Compatibility
XY Stylist is now compatible with the latest iOS version 16.
BUG FIX
Missing promotion on order lines in the Return Cart
We have fixed the bug in the return cart where the item which had a promotion in the initial sale was missing the promotion in the return cart when additional items were added in the return cart with a discount.
Tuesday Oct 6, 2022
Build number: 28.0
ENHANCEMENT
We have added a control on who can change the sales associate post purchase.
Only general managers and cashiers can change the associate post purchase.
Tuesday Oct 4, 2022
Build number: 27.0
ENHANCEMENT
Contextual Menu options
Focus here is to have only the options specific to the page the user is in.
  • The contextual menu in every page has been changed to show only the options specific to that page.
ENHANCEMENT
Improved Usability for open orders preview pods on the home page
The Open Orders pods have been grouped to accommodate showing more information and functionality above the fold on the homepage.
To see all the Open Orders pods > Click on the stack of open orders and then select the order.
ENHANCEMENT
Access Control
We have added a control on who can change the sales associate post purchase.
  • Only general managers and cashiers can change the associate post purchase.
ENHANCEMENT
Alert for a Tax Exempt Order
The user will now get a reminder saying, "For a Tax exempt order the invoice must be generated" on clicking on the Tax Exemption button.
ENHANCEMENT
Alert if product size is not selected in the Product Display Page
If the user clicks on 'Add to Cart' without selecting the size, the system will throw an alert to select the size.
ENHANCEMENT
Receipts
The receipts have been updated to include the item code (style/fabric/color/size).
ENHANCEMENT
Alert if user clicks on the buy button with an item with 0 price the cart
Focus here is to help the sales associates identify if there is an item with 0 price in the cart.
FEATURE
Alert to download the latest version of the app
Focus here is to help the user know that they need to download the latest version of the app.
  • Users with the older version of the app should now see a Pop-up Alert saying "You are on an older version of the App, please update to the latest version"
FEATURE
About page
Focus here is to give the user more information regarding the latest release. The version number from the contextual menu is removed and moved inside the new About page.
  • The user can click on 'About' from the contextual menu which will lead them to the About page which consists of the following information:

    • The release notes for the latest version.
    • The latest app version number.
    • The build version that the user is currently on.

  • We will continue to add more info on the About page based on additional user requests as needed.
BUG FIX
Associate search by partial name in the cart
The user can now find the associate by searching by partial name. For example: Camilla Valerio can be searched by typing 'Camil'.
BUG FIX
Planet Errors
The bug where some planet errors were not showing is now fixed to show all errors.
BUG FIX
Special Characters on notes and invoices
Special characters are now supported on both Notes feature and on the Invoices.
BUG FIX
Cash Drawer
Cash drawer is now configured to open only if the customer pays by cash.
Tuesday Aug 30, 2022
Build number: 26.0
ENHANCEMENT
Improved UI for Discount options
Focus here is to see all the discounts percentages on the same page making it easier now to select the right percentage.
A drop down is now added to add the discount percentage in the Add Discount page.
ENHANCEMENT
Improved UI on order profile page
Focus here is to make sure the order profile page is an accurate recap of the sales/return cart
  • The order profile page now shows if it's a 'Carry Out' order or a 'Return In Store' along with the number of items in that order just like how it was shown in the cart.
  • UPC is added on each item card to help the sales associates when looking into the order profile.
  • Add Note feature is added on the order profile to help the associate add notes after the sale is completed.
  • The associate can now add the Omnichannel Reason on the order profile page after the sale is completed.
ENHANCEMENT
Cash Drawers will be opened for every cash payment made on the fixed POS
Focus here is to enhance the user experience for a cashier
ENHANCEMENT
Products page item transactions
Focus here is to help the associate view all the sales for a particular product.
  • The user can go into any product, click on 'Item Transactions' and click on 'View Sales' to view all the order lines for that product in the current store.
ENHANCEMENT
Cashier name on cards and receipt
Focus here is to add more detail to each transaction
  • Every order will have the cashier name if the sale is closed by the cashier.
  • The Cashier name is now added to the receipt as well.
ENHANCEMENT
Add Discount and Tax Exemption in the Return Cart
Focus here is to let the sales associate add a discount in the return cart in case the item sold during promotions is returned when the promotions have ended.
FEATURE
Order Lines added in the burger menu, Inventory Lookup, Customer Info, User Profile
Focus here is to help the user see all the items sold in one place.
  • Burger menu: On clicking the Order Lines button, all the items sold across all stores can be viewed. The user can filter by My Store, All Orders and My Orders. On clicking on any item, the user will be taken directly to the order profile.
  • Inventory Lookup: Order Lines are now added where the user can view all the order lines for that product across all stores.
  • Customer Info: Order Lines are now added where the user can view all the items sold to the customer across all stores.
  • User Profile:Order Lines are now added under 'Related Links' where the sales associate can view all the items sold by them.
FEATURE
Planet Errors are now shown on Stylist
Focus here is to let the associate know what went wrong in the Planet Tax Free transaction.
  • Planet errors if any will be shown on the same Planet tab on the view confirmation page.
FEATURE
Omnichannel Reason
Focus here is to help the business know through what channel the customer has come to the store.
  • A new feature is added on the cart page under 'Omnichannel Reason'. The user can select the reason by just clicking on the dropdown next to 'Select Reason'.
